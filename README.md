## fpm-plugin-dev-cli
用于创建fpm插件项目的脚手架

### Install
```bash
cnpm i -g fpm-plugin-dev-cli
```

### Useage
```bash
fpm-plugin update
```

it will update the lasted template project from github

```bash
fpm-plugin init sms
```

it will create a project named `fpm-plugin-sms`

then you can dev the project
